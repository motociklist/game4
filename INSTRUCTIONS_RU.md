# Инструкция по добавлению иконки приложения

## ✅ Что уже сделано:

1. ✅ Добавлен пакет `flutter_launcher_icons` в `pubspec.yaml`
2. ✅ Настроена конфигурация для Android, iOS и Web
3. ✅ Создана SVG иконка: `assets/app_icon.svg`

## ⏳ Что нужно сделать:

### Создать PNG иконку (1024x1024)

**Самый простой способ:**

1. Откройте в браузере: https://cloudconvert.com/svg-to-png
2. Загрузите файл: `assets/app_icon.svg`
3. Установите размер: 1024x1024
4. Нажмите "Convert"
5. Скачайте PNG
6. Сохраните как: `assets/app_icon.png`

**Альтернативные способы:**
- Canva: https://www.canva.com/ (создайте 1024x1024, фон #4A90E2, белый "?")
- Paint: Откройте SVG в браузере, сделайте скриншот, обрежьте до 1024x1024

## После создания PNG:

Выполните команды:

```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

Это автоматически создаст иконки для всех платформ:
- ✅ Android (все размеры)
- ✅ iOS (все размеры)
- ✅ Web (favicon и manifest icons)

## Проверка:

После генерации запустите приложение и проверьте иконку на всех платформах.
