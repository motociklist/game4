
Для создания PNG иконки используйте один из методов:

1. ОНЛАЙН (РЕКОМЕНДУЕТСЯ):
   - Откройте https://www.canva.com/
   - Создайте дизайн 1024x1024
   - Фон: #4A90E2 (синий)
   - Добавьте белый текст "?" крупным шрифтом
   - Экспортируйте как PNG
   - Сохраните в assets/app_icon.png

2. ИСПОЛЬЗОВАНИЕ SVG (уже создан):
   - Откройте assets/app_icon.svg в браузере
   - Сделайте скриншот или используйте онлайн-конвертер:
     https://cloudconvert.com/svg-to-png
   - Загрузите SVG, установите размер 1024x1024
   - Скачайте PNG и сохраните как assets/app_icon.png

3. GIMP / Photoshop:
   - Создайте новый файл 1024x1024
   - Залейте фон #4A90E2
   - Добавьте белый текст "?" по центру
   - Экспортируйте как PNG в assets/app_icon.png

После создания PNG запустите:
  flutter pub get
  flutter pub run flutter_launcher_icons
